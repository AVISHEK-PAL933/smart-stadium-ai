# Smart Stadium AI

**The Ultimate AI-Powered Ecosystem for FIFA World Cup 2026**

Smart Stadium AI is a comprehensive, enterprise-grade React Native application designed to manage, monitor, and enhance the entire stadium experience for fans, operations staff, and executives.

## Project Overview
This platform brings together cutting-edge AI features, IoT integrations, and high-performance React Native animations into a single cohesive ecosystem. It is designed to demonstrate how AI can revolutionize live events.

## Key Features
- **AI Match Assistant**: Real-time stats, AI conversational agent.
- **Smart Ticket & Navigation**: AR-style seat navigation and digital ticket wallets.
- **Food Delivery & Merch**: Smart ordering with express lane coordination.
- **Operations & Security**: Crowd intelligence, AI anomaly detection, and emergency routing.
- **Executive Analytics**: Live BI dashboards for revenue and stadium usage.
- **Fan Experience**: Social walls, live polls, reward points, and a digital passport.
- **IoT & Sustainability**: Real-time sensor monitoring for water, power, and climate.

## Technology Stack
- **Framework**: React Native with Expo Router (v57.0.0).
- **Language**: TypeScript (Strict Mode).
- **UI & Animations**: `react-native-reanimated`, Glassmorphism, Material Icons.
- **Charts**: Custom built using Reanimated and SVG (zero dependencies).
- **Architecture**: Domain-driven feature modules.

## Installation & Running Locally
```bash
# Install dependencies
npm install

# Start the Expo development server
npm run web # For web
npm run ios # For iOS simulator
npm run android # For Android emulator
```

## Hackathon Demo Mode
To experience the application in Demo Mode, navigate to **Settings** and toggle the "Live Demo Mode". This populates the app with simulated real-time data, IoT alerts, and live match statistics.

## Project Structure
- `/app` - Expo Router file-based routing.
- `/components` - Reusable UI components (GlassCard, Header, etc).
- `/constants` - Theme, Colors, and Tokens.
- `/data` - Mock data definitions for all modules.
- `/docs` - Detailed architectural documentation.
- `/hooks` - Custom React hooks.
- `/services` - Simulated backend and AI services.
